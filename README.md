## About

GME is a professional financial analysis tool.  
The **backend** is built in **C++**, while the **frontend** is developed using **Python** and **Django**.

This repository primarily focuses on creating **AI-driven diagnostic systems** designed to cover a wide range of areas across the financial sector.  
The goal is to provide robust, scalable, and precise models for comprehensive market, risk, and data analysis.
